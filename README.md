# 🌑 DarkDrop

A minimalist, dark-themed file-sharing platform. Upload files, generate shareable URLs, and manage your profile—all wrapped in a shadowy, user-friendly interface. 🚀

---

## 🛠️ Getting Started

1. **Clone the Repository**  
   ```bash
   git clone https://github.com/SUBHAM-6291/darkdrop.git
   cd darkdrop
   ```

2. **Install Dependencies**  
   ```bash
   npm install
   ```

3. **Launch the App**  
   ```bash
   npm run dev
   ```

Visit [http://localhost:3000](http://localhost:3000) to explore the darkness! 🌙
